# react-form-tools
A set of Tools for building Forms in React

```
<Form fields={[
    {
    type: 'text',
    name: 'user_name',
    placeholder:'Username',
    required: true,
    icon: 'ios-person',
    label: 'Username',
    },
    {
    type: 'passwordChange',
    name: 'password',
    placeholder:'Password',
    icon: 'ios-lock',
    required: true,
    label: 'Password',
    }
]} />
```
